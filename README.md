## 项目目的

- 练习不同设计系统（Material 3、Fluent Design、Neumorphism 等）
- 测试界面交互和视觉效果
- 记录设计迭代过程和灵感来源
- 作为个人学习笔记，方便回顾和分享

## 当前进度

- **2026-01-18**：创建仓库，添加 README + MIT 许可证
- **下一步**：上传第一批 UI Mockup 和 Figma 原型链接
- **计划**：
  - 设计 3-5 个页面原型（登录、首页、详情页）
  - 对比不同配色方案
  - 记录用户反馈（如果有）

## 工具栈

- **设计工具**：Figma、Adobe XD
- **版本控制**：GitHub
- **原型导出**：PNG / SVG / PDF
- **其他**：Coolors（配色）、Fontjoy（字体搭配）
